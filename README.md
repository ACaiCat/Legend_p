# Legend_p
A TShock plugin of role play game.
# 简介
高自由度的RPG插件
主打：公会，技能和高度可自定义的职业
公会：
在公会中你可以和其他玩家互相交易，公会会长有圈定公会领地（tshock自带的region），
任命管理员等权力。
会长和管理员可以踢出或封禁成员。
同公会玩家可相互交易
公会里面玩家可以进行商品交易和购买，同一公会玩家不能攻击同一公会队友
技能：
技能分为三个功能
弹幕，治疗和吸血
超高自由度！
公式算法全部可自定义！
